由于投影仪投影模糊，后排的同学扫不上码的事情时有发生，不得不去前面签到，为了在自己的位置上也能签上到，我们决定开发一款二维码生成器，造福后人。

请注意，**我们不鼓励任何人逃课，这只是一个方便你在教室里签到的工具**！

## 使用方法


1. 在这门课开始考勤之后，你需要得到考勤开始后生成的任意一张二维码
2. 在我们的二维码生成器中选择这张二维码;
3. 随后，你可以扫描生成出来的二维码，它将每隔五秒刷新一次.
